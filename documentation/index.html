Creating your page

Pages

Opening the application first time, you will find landing page where in the center stands a big box with title "start new page".

By clicking on it, an empty canvas will appear with message that a page is empty and you should drop something called contentblocks.

Contentblocks are premade and styled HTML blocks provided by included template (for instace Arkio) that the application comes with and which you you can use on your webpage.

To use contentblocks, click on menu item titled content on the left side.

There you will find list of blocks upon clicking will appear on your page.

Notice: When no page is currently worked on, the only option you can click is download on the left side. To unlock these menu items, start new page, import or load previous page.

Saving page

Page will be automatically saved every minute.

You can also save manually by clicking on Save button on the left navigation tab or by using hotkey Ctrl + R.

Note: Builify allows currently to store 15 pages at maximum. Therefore be careful when you have maximum pages allowed in storage and create a new page which will remove oldest page from stroage to give room to the newest page.

Downloading pages

To download pages, click on download button on the left side.

If you have multiple pages saved, a modal will open where you can select pages what you want to download.

Exporting page

Builify comes with support to export and import pages which for instance is useful for saving work.

To export currently opened page, click on page button on the left side.

Then find button titled export that is at the very bottom and click on it.

You will then download a file with name arkio-page.builify which you can then later import and use.

Importing page

To import page, select load or import page from start screen when no page has been selected and upon clicking on it a modal will open.

Click on import page button in the center of the modal and file explorer will open.

Navigate to place where the page file with .builify extension is located at and double click on it.

Your page will be imported then and it will open in editor which you can edit and save.

Warning: If import file is corrupted or has been manually edited it will be automatically rejected.

Editing your page

How to edit

The application comes with many features to help you at creating web pages.

One of these features are editor tools which there are 3 of them:

  1. Right click editor for elements – a menu that allows you to open certain editors like image and icon changer or edit element. This menu opens when you right click on element when inside page canvas.
  2. Section toolbox for blocks – upon hovering block, you shall notice that in the right corner of the block is a list of icons. The list contains block features such as background color, image, forms and video, which you can edit by clicking on icons.
  3. Block editor – when opening right click editor and clicking on Open Editor, a new panel will appear on the left side. With this panel you can edit selected block. You can change elements opacity, color, rotation, text alignment, line height, letter spacing, classname, margins, paddings and even border radius (applies to certain elements only).

How to change block's background color

To change background color, the block has to support that feature at first place.

For that, hover over block that you want to change and if such paint icon (see picture below) will appear in toolbox then click on it.

Colorpicker will open there and by interacting with it you can change block's background color including opacity by dragging on alpha channel slider.

How to change image sources

If you want to change block's background image, hover on it and if this features is enabled on that block, following icon should appear in toolbox list.

Click on it and modal will open where you can proceed to edit image.

If you want to change image's source within block, right click on it which will open contextmenu and click on Edit Image.

Again, modal will open.

In modal, you will find 4 navigation items:

  1. Library – List of pre selected images from Unsplash that are in different categories which you can pick from.
  2. Current Image – Current image with its preview and source input where you can edit its url or put new url.
  3. Upload new – Here you can upload your own images.
  4. Uploaded images – List of images that you have uploaded and a button Delete all images which will delete all uploaded images.

After doing desired changes, click on save button. To cancel any changes, click on cancel button.

Alternatively to uploading images, you can put your images in assets/images/ folder and put your images there. For example, put a filename called myimage.jpg into assets/images folder. Then in current image tab, click on url inout and change it to assets/images/myimage.jpg. The following image should appear in preview window and thus click save to save changes.

How to change icon

The applications currently supports only FontAwesome icons which come with included template (Arkio).

To change icon, locate it by right clicking on it and contextmenu should open.

Click on edit icon and modal will appear.

On the left side of modal is list of supported icon sets.

Click on icon and the icon will be automatically applied to target element.

Press done to close modal.

Editing forms

Hover over block where form is and following icon should appear in section toolbox.

Click on it and following modal will appear.

The action is path to where form will send information after user sumbits form.

The method has 2 options, whether to POST or GET information.

Use method that suits action url's requested method.
