Template

Overview

The template is based on Bootstrap v4 to provide excellent framework to work upon thus all Bootstrap elements are supported.

To edit HTML manually, use your favourite text editor (Notepad++, Visual Studio Code or advanced editors like WebStorm).

To inspect HTML elements, you can for use browser’s inbuilt tools. Whether using Google Chrome or Mozilla Firefox, right click on element and select inspect element. This will open browser specific editor that will make your life much easier.

File structure

Since Arkio is HTML template, it comes with many HTML (the file extension is .html) files along stylesheets for styling (.css) and JavaScript for interactivity (.js files).

Below is a brief overview of the files at root folder and further details of them.

  • builify

    This folder contains Builify related files and by clicking on index.html, the editor will open in your browser. This is recommend way of building your web page with Arkio. For usage information, forward to official Builify documentation.

  • arkio

    • build

      index.html contains all HTML components. This is useful to have overview of template and can make.

      For individual components look into components/ folder in this folder. There you will find list of all Arkio HTML templates that are categorised in respective folders.

      For Arkio template stylesheet, look into template/template.css.

      For JavaScript file, look into template/javascript.js.

Grid

Since Arkio is based on Bootstrap 4 you can use its awesome 12 column grid system.

For more, read Bootstrap official grid documentation here.

Icons

Arkio officially uses FontAwesome icon pack. Using that icon pack is just a matter of knowing correct class name for desired icon.

You can find the appropriate class names for FontAwesome icon pack here.

To use FontAwesome icon, the element class has to be formated as following:

<i class="fa fa-NAME"></i>

For example, if you want to have address book icon, you write following:

<i class="fa fa-address-book"></i>
